# Terminal Portfolio Website

## Overview
This is my personal portfolio website built using HTML, CSS, and JavaScript.  
The design is inspired by a **terminal/command-line interface**, featuring animated typing effects, glowing UI elements, and an interactive experience section.

The site showcases my background, skills, and experience as a Cybersecurity orriented Computer Science Student

---

## Features
-  Terminal-style UI with typing animations  
-  Animated glowing interface using CSS  
-  Multi-page navigation (Home, Experience, Projects, Contact)  
-  Collapsible experience section (JavaScript-powered)  
-  Social links (GitHub & LinkedIn)  

---

##  Technologies Used
- **HTML5** – Structure of the website  
- **CSS3** – Styling, animations, and layout  
- **JavaScript** – Interactive elements (collapsible sections)  

## How To Run
1. Clone Repository
2. Open project folder
3. Open index.html on your browser (Alternatively install live server extension)
